# Financial KPI

**Version:** `5.1.5-enterprise`  
**API:** `POST /api/finance-rpt/v1/intelligence` with `action: kpi`

## KPI Types

| Type | Example |
|------|---------|
| margin | Gross / operating margin |
| revenue | Growth rate |
| expense | Cost ratio |
| liquidity | Current / quick metrics |
| efficiency | Working capital turns |

Knowledge base: `kpi` under the reporting knowledge graph.
