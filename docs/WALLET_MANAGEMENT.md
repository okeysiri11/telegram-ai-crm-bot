# Wallet Management

**Version:** `5.1.4-enterprise`  
**API:** `GET/POST /api/finance-da/v1/wallets`

## Wallet types

| Type | Use |
|------|-----|
| hot | Operational liquidity |
| cold | Long-term vault |
| multisig | Policy-controlled treasury |
| hd | Hierarchical deterministic derivation |

## Features

Address management · derivation paths · per-wallet balance monitoring · multi-network support (bitcoin, ethereum, tron, bnb, polygon, solana, evm)
