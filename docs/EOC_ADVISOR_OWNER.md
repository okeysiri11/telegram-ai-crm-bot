# EOC — AI Ops Advisor, Release Manager, Incidents & Owner Command

AI Operations Advisor produces a daily platform/pilot report with development recommendations — AI never acts alone; owner approval required.

Release Manager stores release/changelog/migration/test/impact history (via Release Platform).

Incident Center manages errors, critical events, investigations, fixes, and resolution status.

Owner Command Center (platform_owner only): launch companies, promote to Production, approve global changes/releases/mass updates/AI recommendations after Expert Board.
