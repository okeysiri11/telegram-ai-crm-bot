# General Ledger

**Version:** `5.1.0-enterprise`  
**API:** `GET/POST /api/finance-enterprise/v1/ledger`

Chart of Accounts · Double Entry Accounting · Journal Entries · Ledger Posting Engine · Account Balances · Trial Balance

## Actions

| Action | Purpose |
|--------|---------|
| `account` | Add COA account |
| `journal` | Create balanced journal entry |
| `post` | Post journal to ledger |
| `trial_balance` | Generate trial balance |
| `balance` | Fetch account balance |
