# Remote Operations

Remote mission control, ground station, firmware upload, parameter editing, mission upload, log download, diagnostics, camera stream, telemetry, and shell.

## API

- `GET|POST /api/drone/v1/cloud/remote`

Actions: `open`, `command`, `gcs`, `firmware`, `parameters`, `mission_upload`, `log_download`, `diagnostics`, `camera`, `telemetry`, `shell`.
