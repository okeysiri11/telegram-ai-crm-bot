# EOE — Registry & Optimizers

**Sprint:** 24.6 · **API:** `/api/enterprise-eoe/v1`

## Optimization Registry

Each opportunity stores: Opportunity ID, Category, Priority, Business Value, Expected ROI, Confidence Score, Risk Score, Owner Status, Implementation History.

Categories: process · resource · revenue · cost · customer_experience

## Optimizers

| Optimizer | Focus |
|-----------|--------|
| Process | Workflow, CRM, Commerce, Marketing, Finance, Operations, Communications — waste, bottlenecks, delays |
| Resource | Personnel, materials, equipment, working time, branches, AI resources |
| Revenue | Profit, average ticket, repeat sales, LTV, staff load, marketing conversion |
| Cost | Waste spend, unused licenses, inefficient processes, idle resources |
| CX | Journey, operation duration, failure points, repeat contacts, feedback |
