# Visual Automation

Canvas editing (nodes, connections, zoom, grid, mini-map), undo/redo, clipboard, grouping, comments, and flow execution modes (`visual`, `background`, `step`) with breakpoints, logs, retry, and rollback.

## API

- `GET|POST /api/workflow-studio/v1/workflows/{id}/canvas`
- `POST /api/workflow-studio/v1/execute`
