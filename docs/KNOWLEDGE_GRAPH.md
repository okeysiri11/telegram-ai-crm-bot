# Knowledge Graph

Sprint 12.0 merges application knowledge registries into one global graph:

- Platform Core
- AI Ecosystem v1.5
- Unified AI Ecosystem
- CRM / Auto / Agro / Port / Drone
- Knowledge System (`knowledge/`)

## API

- `GET /api/ai-ecosystem/v1/knowledge`
- `POST /api/ai-ecosystem/v1/knowledge` — build/merge graph
