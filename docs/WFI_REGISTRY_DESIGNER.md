# WFI — Registry, Visual Designer & AI Builder

Workflow Registry: ID, name, description, industry, version, status, owner, change history.

Visual Designer (drag & drop): Start, End, Decision, Condition, Delay, AI Decision, Human Approval, API Call, Database, Notification, Payment, CRM, Calendar, Marketing, Commerce.

AI Workflow Builder analyzes bottlenecks, suggests shorter paths, forecasts duration — never mutates workflows alone.
