# EKG — Context Engine, Semantic Search & AI Context Builder

Context Engine assembles CRM, Commerce, Marketing, Workflow, Communications, Product Intelligence, and Operations Center context before AI tasks (target: milliseconds).

Semantic Search understands intent (e.g. inactive VIP clients, high-lift campaigns, booking-related processes).

AI Context Builder sends only relevant entities, related objects, decision history, prior recommendations, and outcomes — filtered by owner AI-allow flags.
