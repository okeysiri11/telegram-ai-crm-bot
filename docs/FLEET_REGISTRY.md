# Fleet Registry

**Version:** `4.5.0-enterprise`  
**Sprint:** 15.0  
**Package:** `applications/port_enterprise/`  
**API:** `/api/port-enterprise/v1`

## Capabilities

- Vessel registry with IMO, flag, ownership, and technical specs
- Shipping lines, carriers, vessel operators, agencies, and service providers
- Current vessel status

## Endpoints

| Method | Path | Purpose |
|--------|------|---------|
| GET/POST | `/fleet` | Register vessel / set status |
| GET/POST | `/shipping` | Line / carrier / operator / agency / provider |
