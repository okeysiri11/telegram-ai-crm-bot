# Connected Cars

**Version:** `4.1.8-enterprise`  
**Sprint:** 13.7  
**Foundation:** Enterprise Platform v4.1.7-enterprise  
**API:** `/api/connected-cars/v1`

Connected Cars ecosystem — telematics, IoT devices, remote diagnostics, predictive maintenance, and real-time fleet intelligence.

## Domains

| Domain | Capabilities |
|--------|----------------|
| Core | Vehicle connectivity/identity, telematics hub, IoT gateway, edge devices, communications |
| Telematics | GPS, trips, routes, driving behavior, fuel/battery, OBD, event recorder |
| Remote | Health, diagnostics, notifications, commands, maintenance alerts, firmware registry |
| Predictive AI | Failure, battery, engine, brake, tire predictions; scheduling; utilization |
| Fleet Intelligence | Dashboard, optimization, driver performance, downtime, fuel efficiency, KPIs |
| Smart Services | Insurance, roadside, charging/fuel/parking registries, service locator |
| Executive Dashboards | Connected fleet, telematics, predictive maintenance, vehicle health |
| Knowledge | Telemetry, fleet, IoT, diagnostics, predictive registries |

Does **not** modify Platform Core, AI OS, Enterprise Edition, or previous Auto Marketplace sprint packages.
