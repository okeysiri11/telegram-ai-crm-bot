# Financial Modeling

**Version:** `5.1.6-enterprise`  
**API:** `GET/POST /api/finance-cfo/v1/modeling`

## Models

| Type | Purpose |
|------|---------|
| roi | Return on investment |
| npv | Net present value |
| irr | Internal rate of return |
| break_even | Units / volume to break even |
| sensitivity | Driver shock impact |
| what_if | Scenario uplift / downside |
