# Enterprise Architecture

**Version:** `5.3.0-enterprise`  
**Sprint:** 19.0  
**Package:** `applications/enterprise_hub/`

## Layers

1. **Enterprise Registry** — platforms, services, modules, integrations, organizations, environments  
2. **Integration Layer** — discovery, gateway, routing, ESB, aggregation  
3. **Enterprise Identity** — unified identity, org/user/role mapping, permission sync  
4. **Enterprise Configuration** — global config, feature flags, platform settings, profiles  
5. **Event Infrastructure** — event types, bus, routing, logs, replay, DLQ  
6. **Executive Dashboard & Knowledge** — overview and graph registries  

Additive hub atop frozen vertical platforms (Finance 5.2.0, Legal 5.0.0, Crypto 4.8.0, Port 4.6.0, Agro 4.4.0, Automotive 4.2.0).
