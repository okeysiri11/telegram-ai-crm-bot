# Portfolio Management

**Version:** `4.7.0-enterprise`  
**Sprint:** 16.0  
**Package:** `applications/crypto_enterprise/`  
**API:** `/api/crypto-enterprise/v1`

## Capabilities

- Portfolio and wallet registries
- Asset allocation and PnL tracking
- Balance history snapshots
- Asset registry: coins, tokens, blockchains, stablecoins, pairs
- Dashboards: exchange, portfolio, market, asset

## Endpoints

| Method | Path | Purpose |
|--------|------|---------|
| GET/POST | `/portfolio` | Portfolios, wallets, allocation, PnL, balances |
| GET/POST | `/assets` | Coins, tokens, chains, stablecoins, pairs |
| GET/POST | `/dashboard` | Exchange / portfolio / market / asset dashboards |
| GET/POST | `/knowledge` | Crypto knowledge graph registries |
