# Navigation (Resilient)

Sprint **11.9** — multi-source navigation with GPS, RTK, visual navigation, dead reckoning, and position confidence.

## Capabilities

- Navigation Manager & Health Monitor
- Multi-source Position Estimator
- GPS / RTK Managers
- Visual Navigation Interface
- Dead Reckoning
- Position Confidence Estimator

## API

- `GET|POST /api/drone/v1/resilience/navigation`

Also see legacy `GET /navigation` and `POST /navigation/plan` (Sprint 11.4).
