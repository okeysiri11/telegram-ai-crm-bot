# Crypto Enterprise Deployment

**Version:** `4.8.0-enterprise`

## Deploy

1. Apply `applications/crypto_enterprise/release/config.template.env`
2. Register `register_crypto_enterprise_routes`
3. Confirm all API healthchecks
4. Run certification bootstrap
5. Follow `PRODUCTION_CHECKLIST.md`
