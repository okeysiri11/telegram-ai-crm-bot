# Administrator Guide

- Monitor certification and readiness dashboards (`enterprise_health`, `certification`, `performance`, `security`, `release`).
- Run periodic `POST /api/legal-enterprise-certification/v1/bootstrap` after configuration changes.
- Keep peer platforms frozen; do not patch Platform Core / AI OS / Enterprise Edition from Legal.
- Use production and operations checklists during go-live.
- Escalate critical legal alerts via Executive Intelligence alert center.
