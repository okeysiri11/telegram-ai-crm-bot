# Operations Manual — Agro Enterprise Suite v4.4.0-enterprise

## Daily

- Check certification and module health endpoints
- Review executive briefing / readiness score if used with AI Agronomist

## Weekly

- Re-run performance and security certification gates
- Validate documentation and release artifact presence

## Incident response

1. Isolate failing module API prefix
2. Capture certification scorecard
3. Roll back using prior Agro Enterprise release package if required
4. Re-run QA regression suite after fix

## Backups

Persist store snapshots and configuration templates with each production cutover.
