# Administrator Guide — Port Enterprise Suite v4.6.0-enterprise

- Use `X-Principal` for authenticated requests.
- Keep module API prefixes isolated; do not collapse routes.
- Run certification after configuration changes.
- Monitor executive readiness dashboard (`enterprise_readiness`).
- Preserve frozen platforms: AI OS `3.4.0-alpha`, Enterprise `4.0.0-enterprise`, Auto `4.2.0-enterprise`, Agro `4.4.0-enterprise`.
- Rotate secrets via environment variables from `config.template.env`.
